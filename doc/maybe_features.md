# Maybe Features Registry

This file tracks features that are proposed and marked as "maybe" for future consideration.

## 1. Feature 9: Voice-Keyed Cryptographic Scrambling (Secure Tactical Comms)
* **Status:** Maybe
* **Description:** For tactical squads, introduce a "Secure Comm Mode" where a private cryptographic key is shared. When enabled, the voice stream is modulated/scrambled on the wire and decrypted only by allies with the same key. To anyone else scanning or intercepting the frequency (or listening nearby in proximity without a key), it sounds like unintelligible digital noise or sci-fi encrypted garble, preventing eavesdropping during stealth operations.
